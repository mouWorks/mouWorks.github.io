## Try at Local

### Req: `@mac`
* Virtualbox
* Minikube
* kubectl

### Trying with Minikube
* After installation:
    * `minikube start` -> 啟動 minikube
* Minikube conf file loaction
    * `~/.kube/config`    
* Minikube commands:
    * `where minikube`: 查看本機安裝的位置
    * `minikube version`: 看版本
    * `minikube status`: 看狀態
    * `minikube start`: 啟動 minikube cluster
