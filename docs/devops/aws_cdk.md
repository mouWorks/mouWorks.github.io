# AWS CDK

* 用來建置 AWS Stack 環境的
* 程式版本的 `CloudFormation`

### 支援語言

* TypeScript
* Java
* JavaScript
* Python

### Steps

#### Install

* Pre-req 需要先設定以下
    * `awscli`
    * `awscli` 相關的 credential

* Install AWS CDK 
    * `npm install -g aws-cdk`
* command
    * `cdk --version`    
    
#### Staring

* Init the repo
    * `cdk init --language javascript`
