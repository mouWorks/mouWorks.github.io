# 列出常用的 LeetCode 算法

### 遍歷
* 最簡單的, for-loop 全部跑一遍, 最 easy 的情況下跑一遍 O
```
for (let i = 0; i < 3; i++) {
  console.log ("Block statement execution no." + i);
}
```

* 但最常發生的事需要 兩層 for-loop O^2 -> 這通常是要避免的情況
