# Steps

# Install

  • Pre-req 需要先設定以下

    • awscli
    • awscli 相關的 credential

  • Install AWS CDK

    • npm install -g aws-cdk

  • command

    • cdk --version

# Staring

  • Init the repo
    • cdk init --language javascript
